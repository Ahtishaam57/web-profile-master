AutoCAD Project

Building Floor Plan Project As the final project of my first semester, I independently designed a comprehensive building floor plan using AutoCAD. This project included various elements such as the main building structure, parking spaces, driveways, trees, and bicycle parking areas. Key features of the project: Detailed Floor Plan: A precise layout of the building's interior and exterior spaces, ensuring all measurements were accurate. Parking and Driveways: Thoughtfully designed parking spaces and driveways to optimize traffic flow and accessibility. Green Spaces: Integration of trees and landscaped areas to enhance the aesthetic and environmental quality of the site. Bicycle Parking: Designated bicycle parking areas to promote eco-friendly transportation options. Throughout this project, I meticulously calculated all dimensions and lengths, ensuring that every aspect of the plan adhered to real-world standards and specifications. This project not only enhanced my technical skills in AutoCAD but also honed my ability to manage and execute complex design projects independently.

Sanitary Sewer System Design and Analysis

Sanitary Sewer System Design and Analysis In this project, I was responsible for designing a sanitary sewer system. The design includes the following elements: PVC Pipe Layout: The system uses 250mm diameter PVC pipes, with detailed sections showing the length and slope of each segment. For example, one segment measures 92.8 meters with a slope of 1.14%, ensuring efficient wastewater flow. Manhole Placement: Manholes are strategically placed at various points (e.g., MH1, MH1A, MH2) to facilitate maintenance and access. These manholes are crucial for the inspection and cleaning of the sewer system. Elevation and Coordinates: The design incorporates precise coordinates and elevation points for the start and end of each sewer section. This precision is necessary to maintain the correct slope and flow direction. Slope Calculation: Proper slope calculation is essential for the functionality of the sewer system. Slopes are calculated to ensure gravity flow, preventing any backflow or clogging issues. This design demonstrates my ability to apply engineering principles to create an efficient and maintainable sanitary sewer system, highlighting my attention to detail and technical skills in civil engineering projects.

Roof Forming Plan

This project involved developing structural drafting skills using AutoCAD to create a Roof Framing Plan for a typical unit in the Red Earth Meadows Subdivision. The assignment required the use of specific AutoCAD functions and drafting conventions to replicate real-world structural drafting projects. Key tasks included copying and aligning gridlines, creating and blocking grid bubbles, drafting detailed connection views, and organizing a Roof Beam Pre-Cut Schedule. The project emphasized accuracy, attention to detail, and adherence to professional drafting standards, culminating in the creation of detailed PDF plans and an AutoCAD file submission.
